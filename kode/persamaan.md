# Menyelesaikan persamaan matematika

Soal:

2x + 2y + 2z = 2

Kode:

```coffeescript
# 2x + y + 2z = 2

mulai = -5
akhir = 5

for x in [mulai..akhir]
  for y in [mulai..akhir]
    for z in [mulai..akhir]
      if 2 * x + y + 2 * z is 2
        console.log "Jawaban: #{x}, #{y}, #{z}"
```
